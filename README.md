# Лабораторні роботи з Haskell Розроблено студентом ІПС-42 Чорнокуром Іваном

Колекція лабораторних робіт з програмування на мові Haskell.

## Lab 1 Варіант 7: Інформаційна система "Спорт на факультеті "

Програма для управління інформацією про спортивні секції та змагання на факультеті з використанням MySQL.

**Особливості:**
- Використання type classes та instances
- Підключення до MySQL бази даних
- 8 таблиць БД (студенти, викладачі, секції, розклади, змагання тощо)
- CRUD операції для всіх сутностей

**Запуск:**
```bash
cd lab1
cabal run
```

## Lab 2 Варіант 1: Багатопотокове стиснення файлів алгоритмом Хаффмана 

Багатопотокова програма для стиснення множини файлів в один архів з використанням алгоритму Хаффмана.

**Особливості:**
- Алгоритм Хаффмана для стиснення
- Багатопотокова обробка з GHC `-threaded`
- Паралельне стиснення множини файлів
- Бенчмарки для вимірювання продуктивності

**Запуск:**
```bash
cd lab2
cabal run huffman-compression -- compress-parallel archive.huf
cabal run huffman-compression -- benchmark
```

## Lab 3 Варіант 7: Обробка тексту підручника 

Програма для обробки тексту підручника з використанням типізованих структур даних.

**Особливості:**
- Типи: Символ, Слово, Пропозиція, Розділовий знак
- Нормалізація пробілів (заміна табуляцій одним пробілом)
- Сортування слів за зростанням частки голосних букв
- Статистика тексту

**Запуск:**
```bash
cd lab3
cabal run text-processor -- textbook.txt
```

Для кожної лабораторної роботи:
